# Metric Reference

The benchmark records diff, token, timing, complexity, and behavior metrics for each task run.
Diff metrics capture changed files and line deltas.
Token metrics capture input tokens, output tokens, total tokens, and estimated cost when pricing is configured.
Timing metrics record wall clock seconds.
Complexity metrics capture tool calls, prompt size, workspace C# file counts, and syntax node counts.
Behavior metrics capture exit code, success flag, and output size.
